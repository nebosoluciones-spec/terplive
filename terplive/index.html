How to capture system audio

Transcribe audio from Zoom, Teams, Interprefy or any other app — without a physical microphone.

💡 Recommended setup
Keep TerpLive in a separate Chrome tab from your interpretation platform (Zoom, Teams, Interprefy, etc.). This way you can switch between them quickly without interrupting the transcription.

Important: For your colleagues and clients to hear you in the interpretation session, make sure your headset microphone is selected as the input device — either in Chrome's site settings or directly in the app you use (Zoom, Teams, etc.). TerpLive uses a virtual audio driver to capture incoming audio, but your headset mic should remain the output device for your voice.
Mac — BlackHole
Free virtual audio driver
1
Download BlackHole 2ch — go to existential.audio/blackhole and download the free installer.
2
Install it — run the installer and restart your Mac when prompted.
3
Open Audio MIDI Setup — press Cmd+Space and search "Audio MIDI Setup".
4
Create a Multi-Output Device — click the + button at the bottom left → "Create Multi-Output Device". Check both BlackHole 2ch and your headphones or speakers. This lets you hear the audio AND capture it at the same time.
5
Set output — go to System Settings → Sound → Output → select Multi-Output Device.
6
In TerpLive — click the 🎤 microphone button. When Chrome asks for microphone access, select BlackHole 2ch as the input device. Done — TerpLive will now transcribe all system audio.
💡 To switch back to your regular microphone, go to System Settings → Sound → Output and select your normal speakers or headphones.
🪟
Windows — VB-Cable
Free virtual audio cable
1
Download VB-Cable — go to vb-audio.com/Cable and download the free driver.
2
Install it — run as Administrator and restart Windows.
3
Set playback output — right-click the speaker icon in the taskbar → Sound Settings → Output → select CABLE Input (VB-Audio).
4
Keep hearing audio — in Sound Settings → search "App volume" → set your browser's output to your headphones so you can still hear while capturing.
5
In TerpLive — click the 🎤 button. When Chrome asks for microphone access, select CABLE Output (VB-Audio). TerpLive will now transcribe all system audio.
💡 To switch back to your microphone, go to Sound Settings → Output and select your normal device.
